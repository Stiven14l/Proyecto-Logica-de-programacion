# Proyecto-Logica-de-programacion
Juego "Adivina el Número"

28 de Junio del 2026

Objetivo del Proyecto

Desarrollar un programa interactivo en Python que permita al usuario adivinar un número generado aleatoriamente por el sistema, aplicando los conocimientos adquiridos en la asignatura de Lógica de Programación, tales como variables, estructuras condicionales, ciclos repetitivos, funciones, estructuras de datos y manejo de excepciones.

Descripción

El proyecto "Adivina el Número" es un juego desarrollado en Python en el que el usuario debe descubrir un número secreto generado aleatoriamente por el sistema.

El programa ofrece un menú principal desde el cual el usuario puede iniciar una partida, consultar el historial de juegos, visualizar la mejor puntuación obtenida y salir del programa.

Durante cada partida el sistema proporciona pistas indicando si el número ingresado es mayor o menor que el número secreto, facilitando que el jugador llegue a la respuesta correcta.

Funcionalidades
Menú principal.
Tres niveles de dificultad (Fácil, Medio y Difícil).
Generación aleatoria del número secreto.
Validación de datos mediante manejo de excepciones.
Registro del nombre del jugador.
Contador de intentos.
Historial de partidas utilizando listas y diccionarios.
Consulta de la mejor puntuación.
Interfaz de consola sencilla y fácil de utilizar.
